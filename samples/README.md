# Sample Textbooks

Place sample PDF textbooks here for testing.

Recommended: Download any Maharashtra State Board Science textbook (Std 6-12) from:
- https://cart.ebalbharati.in/BalBooks/ebook.aspx
- Choose Science, English Medium, any standard

Or use any school textbook PDF you have available.

The system supports:
- Text PDFs (direct text extraction)
- Scanned/image PDFs (OCR with EasyOCR - Hindi, Marathi, English)
- Maximum file size: 50MB
